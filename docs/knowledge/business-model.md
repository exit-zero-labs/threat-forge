# ThreatForge — Business Model

## Revenue Model

ThreatForge is open-source and donation-funded. Revenue is not the goal; sustainability is.

| Source | Mechanism | Estimated Annual (Year 2+) |
|--------|----------|---------------------------|
| GitHub Sponsors | Individual $5-25/mo | $5,000-$30,000 |
| Corporate sponsorships | Logo placement, support | $10,000-$50,000 |
| Open Collective | One-time + recurring | $2,000-$10,000 |

**Total estimated Year 2 funding:** $20,000-$100,000 — wide range reflecting open-source funding uncertainty.

## Sustainability Milestones

| Milestone | Metric | Significance |
|-----------|--------|-------------|
| Self-sustaining infrastructure | Sponsors cover $600/year costs | Infrastructure covered |
| Bounty-funded development | $500+/month in sponsorships | Can fund community bounties |
| OWASP project status | Official acceptance | Organizational backing, credibility |

## Path to Sustainability

1. **Months 0-6:** Solo development. Zero external funding needed. Ship and launch.
2. **Months 6-12:** Community forms. Open GitHub Sponsors. First corporate users.
3. **Months 12-24:** If adoption >10K users, apply for OWASP project status. Seek corporate sponsorships.
4. **Long-term:** If 50K+ users, explore managed cloud service (like Excalidraw+) as optional paid offering, keeping desktop app fully free.
